HƯỚNG DẪN CHẠY CODE DEMO
Có thể tải source code về và chạy trực tiếp trên Unity hoặc mở file build để chạy game
Khi chạy game:
+ Khởi động game
	-B1: Nhấn Start game để bắt đầu trò chơi
	-B2: Chọn level màn chơi muốn chơi. 
	
+ Game-Play:
	-B1: Click chọn trụ phòng thủ ở bên phải màn hình 
	-B2: Đặt trụ xuống các ô màu trắng vị trí dành cho trụ
	-B3: Nhấn vào các trụ đã xây để hiện lên UI nâng cấp/bán trụ
		.Nhấn vào nút Upgrade để nâng cấp trụ (tối đa 3 cấp)
		.Nhấn vào nút Sell để bán trụ
	-B4: Khi máu (HP) của người chơi trở về 0 thì sẽ xem như là loose game.
	-B5: Trong khi chơi có thể nhấn phím P để dừng trò chơi.
		.Khi UI dừng trò chơi hiện ra nhấn continue để tiếp tục chơi 
		.Nhấn Retry để chơi lại màn chơi đó. 
		.Nhấn Main Menu để trở về Menu chính.
	-B6: Khi thắng màn chơi:
		.Nhấn continue để mở khóa và chuyển sang màn chơi tiếp theo.
		.Nhấn Main menu để mở khóa màn chơi tiếp theo và trở về màn hình chính
		.Nhấn Retry để chơi lại màn chơi đó
	-B7: Khi thua màn chơi:
		.Nhấn Retry để chơi lại màn chơi đó  
 		.Nhấn Main Menu để trở về Menu chính.
	-B8: Nhấn phím ESC để khóa di chuyển camera.
+Cửa hàng:
	-B1: Bấm chọn Shop ở menu chính
	-B2: Khi vào giao diện cửa hàng nhấn chọn trụ muốn mua, trụ sau khi mua sẽ trừ kim cương và xóa khỏi cửa hàng
	-B3: Nhấn Back để trở về Menu chính.
+Settings:
	-B1: Nhấn vào nút Settings 
	-B2: Nhận vào chọn tỉ lệ khung hình
	-B3: Nhấn vào chọn độ phân giải
	-B4: Nhấn vào thanh trượt để tăng giảm âm lượng trong game
+Thoát khỏi game:
	-Nhất Quit ngoài menu chính để thoát khỏi game
